      • CanJS

      page

      CanJS is an evolving and improving set of client side JavaScript architectural libraries that balances innovation and stability.

      CanJS targets experienced developers building complex applications with long futures ahead of them. Read more about CanJS's mission and goals and why you should give it a try.
